Privacy Policy

Last updated: May 12, 2026

1. Information We Collect

Our application accesses social media platform APIs solely for the purpose of publishing content to our own accounts. We do not collect, store, or process any personal data from end users or viewers.

2. Use of Data

We use platform APIs (YouTube, TikTok, Facebook, Instagram) exclusively to upload and manage our own video content. No user data is collected, shared, or sold.

3. Third-Party Services

Our application integrates with the following services for content creation and distribution:

  • YouTube Data API - for video uploads
  • TikTok Content Posting API - for short-form video posting
  • Facebook Graph API - for page content posting
  • Pexels API - for stock footage
  • HuggingFace API - for AI-generated visuals

4. Data Storage

Authentication tokens for social media platforms are stored locally on our secure server and are never transmitted to third parties.

5. Data Sharing

We do not share, sell, or distribute any data to third parties.

6. Security

We implement reasonable security measures to protect stored credentials and application data.

7. Children's Privacy

Our application is not directed at children under 13 and we do not knowingly collect data from minors.

8. Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected on this page with an updated date.

9. Contact

For privacy concerns, contact us through our social media channels.
